Creación de la imagen de MySql Ventas
$ docker build -t mysql_ventas .

Comando para ver el historial de la imagen creada
$ docker history <id_imagen>

Crear el contenedor de la imagen
$ docker run -d -p 3307:3306 --name bd_ventas denishuaman/mysql_ventas

Construir los jar de las APIs

apiPersona
$ mvn -f apiPersona/pom.xml clean package -Dmaven.test.skip=true

apiVenta
$ mvn -f apiVenta/pom.xml clean package -Dmaven.test.skip=true


Crear un manager
docker-machine ssh manager1 "docker swarm init --advertise-addr $(docker-machine ip manager1)"
